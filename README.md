# Alexaurio----SA-PLD3-Ejercicio-3

Calculadora de Factorial

Descripción
Este proyecto es una página web que calcula el factorial de un número ingresado por el usuario.
El resultado se muestra en pantalla y también en la consola.

¿Cómo funciona?

El usuario hace clic en el botón.
Se le pide un número usando una ventana (prompt).
El programa verifica que el número sea válido (mayor o igual a 0).
Calcula el factorial usando un ciclo.
Muestra el resultado en la página.


¿Qué es el factorial?
Es la multiplicación de todos los números desde 1 hasta el número ingresado.
Ejemplo:
5! = 1 × 2 × 3 × 4 × 5 = 120


Tecnologías usadas

HTML
CSS
JavaScript


Validaciones

Solo acepta números
No permite números negativos
Permite cancelar la operación


Ejemplos

5 → 120
6 → 720
0 → 1
